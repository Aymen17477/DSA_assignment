1. Stack – Reverse a Word

This program uses a stack to reverse a given word.
Characters are pushed and popped to display the reversed output.

2. Balanced Parentheses

This program checks whether an expression has balanced parentheses.
It uses a stack to match opening and closing brackets.

3. Next Greater Element (Stack)

This program finds the next greater element for each array element.
A stack is used to efficiently compare elements from right to left.

4. Printer Queue (Queue)

This program simulates a printer queue using FIFO concept.
Documents are added, printed, and displayed in order.

5. Circular Queue

This program implements a circular queue using an array.
It allows enqueue, dequeue, peek, and display operations efficiently.
